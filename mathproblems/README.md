# mathproblems

Central folder for **all math-problem solutions and docs** (task 159).

Every numbered problem from `mathdot problems.txt` that gets solved keeps its
artifacts here:

- the **solution code** (a mathdot `.md` notebook, or a MATLAB/Mathematica file
  when the problem specifies that tool),
- any **plots** produced, and
- exactly **one documentation `.md`** per problem.
